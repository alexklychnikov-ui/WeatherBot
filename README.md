# Zerocode API - Тестовый модуль для HTTP запросов

Набор инструментов для тестирования API запросов с красивым выводом результатов в консоль.

## Структура проекта

```
API/
├── http_client.py      # Базовый HTTP клиент
├── test_api.py         # Тестовый модуль для API запросов
├── country_info.py     # Модуль для получения информации о странах
├── requirements.txt    # Зависимости проекта
└── README.md          # Документация
```

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Модули

### http_client.py
Базовый HTTP клиент с функциями:
- `get(url, params, headers, timeout)` - GET запрос с проверкой статуса
- `post(url, data, json_data, headers, timeout)` - POST запрос
- `get_simple(url, params, headers, timeout)` - GET без автопроверки статуса

### test_api.py
Интерактивный тестовый модуль для выполнения HTTP запросов:

```bash
python test_api.py
```

**Доступные функции:**
- **0** - Выход из программы
- **1** - Произвольный GET запрос
- **2** - Произвольный POST запрос  
- **3** - Информация о стране
- **4** - Случайное фото собаки по породе
- **5** - Информация о птице (демо)

**Примеры использования:**

GET запрос:
```
Выберите тип запроса (0-5): 1
Введите URL: https://httpbin.org/get
Заголовки (JSON формат, Enter для пропуска): {"User-Agent": "TestApp"}
Параметры (JSON формат, Enter для пропуска): {"test": "value"}
```

POST запрос:
```
Выберите тип запроса (0-5): 2
Введите URL: https://httpbin.org/post
Заголовки (JSON формат, Enter для пропуска): 
Тип данных (1-JSON, 2-form-data, Enter для пропуска): 1
JSON данные: {"name": "test", "value": 123}
```

Информация о стране:
```
Выберите тип запроса (0-5): 3
Введите название страны: germany
```

Фото собаки:
```
Выберите тип запроса (0-5): 4
Популярные породы: beagle, bulldog, husky, labrador, poodle, retriever
Введите породу собаки (Enter для случайной): husky
```

### country_info.py
Красивый вывод информации о странах с цветным форматированием:

```bash
python country_info.py
```

**Возможности:**
- Поиск стран по названию
- Цветной вывод с эмодзи
- Подробная информация: столица, население, языки, валюты и др.
- Циклический ввод с возможностью выхода

**Пример использования:**
```
Введите название страны (или 'exit' для выхода): russia
```

## API endpoints

Модуль использует следующие публичные API:

- **REST Countries** - информация о странах: `https://restcountries.com/`
- **Dog CEO** - фотографии собак: `https://dog.ceo/`
- **JSONPlaceholder** - тестовые данные: `https://jsonplaceholder.typicode.com/`

## Особенности

- **Обработка ошибок** - корректная обработка сетевых ошибок и неверных ответов
- **Цветной вывод** - использование colorama для Windows совместимости
- **Валидация данных** - проверка пород собак через API
- **Гибкие параметры** - поддержка заголовков, параметров, JSON и form-data
- **Таймауты** - защита от зависших запросов

## Системные требования

- Python 3.6+
- Windows 10 Pro (тестировалось)
- Интернет соединение для API запросов

## Зависимости

- `requests` - HTTP клиент
- `colorama` - цветной вывод в консоли
